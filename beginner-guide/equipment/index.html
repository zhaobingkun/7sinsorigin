Seven Deadly Sins Origin SSR Equipment Guide

A compact SSR-only equipment list from your provided sheet with base stats and set-effect text.

Quick Build Notes

  • All entries on this page are SSR grade.
  • Current source set shares the same base stat line: HP 14739 and DEF 5276.
  • Prioritize effects that match your rotation: cooldown cuts, crit-based triggers, and DoT control.

SSR Snapshot

  • Cooldown focus: Royal Dignity, Prince's Adventure
  • Burst scaling: Prince's Adventure
  • Control synergy: Sword of the Old King
  • Sustain utility: Adventure's Beginning

Detailed SSR Equipment List

Royal Dignity

SSR · Base HP 14739 · Base DEF 5276

Set Effect: Burn Mark

Attacking an enemy with reduced Fire Resistance decreases the Normal Skill cooldown by 50% and increases Fire Burst Efficiency by 20% for 15 sec. (Cooldown: 20 sec)

Prince's Adventure

SSR · Base HP 14739 · Base DEF 5276

Set Effect: Growing Ember

Using the Normal Skill while Normal Skill damage boost is active decreases the Special Attack cooldown by 60% and increases Ultimate Move damage by 100% for 15 sec. (Cooldown: 20 sec)

Daughter of the Forest and Earth

SSR · Base HP 14739 · Base DEF 5276

Set Effect: Pre-heat

Each hit on an enemy affected by Combustion decreases their Crit Resistance by 1% for 15 sec. (Max: 30%). When maxed, the enemy's Normal Skill damage taken increases by 10% for 15 sec. (Cooldown: 20 sec)

Adventure's Beginning

SSR · Base HP 14739 · Base DEF 5276

Set Effect: Heat of Support

Increases Healing Efficiency by 40%. Using the Special Attack restores the Tag Gauge by 600. (Cooldown: 15 sec)

Sword of the Old King

SSR · Base HP 14739 · Base DEF 5276

Set Effect: Skull Smash

Increases damage dealt to Stunned enemies by 25%. Landing the Special Attack on a Stunned enemy increases the Stun duration by 3 sec. (Cooldown: 20 sec)

Secret Mission

SSR · Base HP 14739 · Base DEF 5276

Set Effect: Tomato Festival

If an attack on an enemy taking DoT results in a Critical Hit, the DoT interval decreases by 16% for 10 sec. (Cooldown: 15 sec, does not stack)
