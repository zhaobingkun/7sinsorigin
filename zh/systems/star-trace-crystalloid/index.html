七大罪：Origin Star Trace Crystalloid 指南

Star Trace Crystalloid 重要的原因，不只是它在商店里有一个分页，而是它已经进入英雄成长、Potential Point 和版本问题修正的交叉点。官方也已经把它单独列进 1.5 版本议题里。

官方已经确认了什么

  • Developer Notes #13 明确把 Star Trace Crystalloid Issues 列为 1.5 版本重点话题之一。
  • 更早的更新说明也显示，特定英雄的 Potential Point 会放在 Star Trace Crystalloid 商店分页出售。
  • 这说明它不是边角系统，而是已经进入真实成长判断里的核心小系统。

适合什么时候看

如果你现在的问题不只是“这是什么货币”，而是“它会不会影响我当前角色投资和商店决策”，这页就该先看。

最先该重看的 3 件事

  • 你当前把 crystalloid 相关资源投给的英雄，是否真的是当前最值得投的人
  • 你现在的商店选择，是在帮当前 roster，还是只是在追热度
  • 1.5 版本调整后，你现在的 Potential Point 花法会不会反而变得不够稳

适合搭配的页面

建议和 成长与强化系统、Daisy 角色页、Escanor 角色页、Developer Notes #13 一起看。

为什么这页值得单独做

玩家会搜索这条系统，通常不是因为名词本身，而是因为一个成长货币开始真实影响英雄判断。这里已经是这种情况了：商店分页、Potential Point 和官方问题追踪都说明它已经进入 live-service 规划层面。
